---
title: ¿Qué es ALIVIA?
---

ALIVIA es una plataforma de votación digital que combina **Blockchain (Syscoin NEVM)** con **Inteligencia Artificial** para garantizar procesos electorales seguros, transparentes y auditables.

## Principios fundamentales

- **Inmutabilidad**: Cada voto se registra como una transacción en Syscoin NEVM. Una vez confirmado, no puede ser modificado ni eliminado.
- **Transparencia**: Cualquier ciudadano puede verificar los resultados en el explorador de bloques sin permisos especiales.
- **Privacidad**: El voto es secreto gracias a cifrado de extremo a extremo y pruebas de conocimiento cero (ZKP).
- **Un voto por persona**: Verificación biométrica con IA combinada con documento de identidad.

## Stack tecnológico

| Componente | Tecnología |
|---|---|
| Frontend | React + Wasp + Vite |
| Backend | Node.js + Wasp + Prisma |
| Base de datos | PostgreSQL 16 |
| Blockchain | Syscoin NEVM |
| Contenedores | Docker + docker-compose |
| Proxy | nginx-proxy |
| SSL | Let's Encrypt (DNS-01) |
